# Quick Start:
* `pip install --editable .`
* `export FLASK_APP=itupass`
* `export FLASK_DEBUG=1` (optional)
* `export SECRET_KEY=Not#So@Secret` (optional)
* `export DATABASE_URI=postgresql://username@password@server.domain:5432/database`
* `flask initdb`
* `flask run`
